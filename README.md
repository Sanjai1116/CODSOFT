# CODSOFT - Python Programming Internship

This repository contains all the tasks completed during my **Python Programming Internship** at CodSoft.

## Tasks Completed

### Task 1 - To-Do List
A command-line application to create, update, and manage daily tasks.

### Task 2 - Calculator
A simple calculator that performs basic arithmetic operations (addition, subtraction, multiplication, division).

### Task 3 - Password Generator
A tool that generates strong, random passwords based on user-specified length, using a mix of letters, numbers, and symbols.

### Task 4 - Rock-Paper-Scissors Game

An interactive game where the user plays Rock-Paper-Scissors against the computer, with random computer choices and win/lose/tie logic.

### Task 5 - Contact Book

A contact management application to add, view, search, update, and delete contacts with name, phone number, email, and address.
